# AI News Bias Dashboard

This Streamlit app visualizes AI-related incidents, focusing on bias, misinformation, and ethical risks.

## How it works

## Run locally
```bash
# 1) Install
pip install -r requirements.txt

# 2) Run the app
streamlit run app.py
```
